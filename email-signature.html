Instrucciones de instalación

  1. Gmail: Configuración → Ver todos los ajustes → General → Firma → Crear nueva → pegar código en modo HTML
  2. Outlook: Archivo → Opciones → Correo → Firmas → Nueva → editor HTML
  3. Apple Mail: Preferencias → Firmas → crear nueva → editar con HTML via Editar → Adjuntos
  4. Las imágenes ya usan URLs absolutas del dominio live — no necesitas cambiar nada.
  5. Si la foto aparece sin recortar en Outlook, usa una versión pre-recortada a 150×150 px cuadrada y súbela al servidor con otro nombre.
